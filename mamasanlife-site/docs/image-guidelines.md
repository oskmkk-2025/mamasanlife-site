# デザインガイドライン：アイキャッチ画像（折り紙・切り絵風）

今後の記事作成において、サイトの「洗練された大人向けのミニマルデザイン」を維持するための画像生成指針です。

## スタイル定義
- **技法**: 折り紙（Origami）、切り絵（Kirie / Paper-cut）
- **質感**: 和紙のような質感（Japanese paper texture）
- **背景**: 柔らかなパステルカラー、または質感のある白/クリーム
- **雰囲気**: 清潔感、洗練（Sophisticated）、ミニマル（Minimalist）、大人可愛い（Mature Cute）

## プロンプトのテンプレート
