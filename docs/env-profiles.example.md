# 環境変数プロファイル例（Vercel推奨）

## Preview（切替前・検証用）
```
NEXT_PUBLIC_SITE_URL=https://<your-preview>.vercel.app
NEXT_PUBLIC_NOINDEX=true
NEXT_PUBLIC_GA_ID=
NEXT_PUBLIC_ADSENSE_ID=

NEXT_PUBLIC_SANITY_PROJECT_ID=xxxx
NEXT_PUBLIC_SANITY_DATASET=production
SANITY_PROJECT_ID=xxxx
SANITY_DATASET=production
SANITY_API_VERSION=2024-03-14
SANITY_READ_TOKEN=（必要なら）
SANITY_WRITE_TOKEN=（必要なら）

LINE_BROADCAST_ENABLED=false
LINE_CHANNEL_ACCESS_TOKEN=
SANITY_WEBHOOK_SECRET=
CRON_SECRET=dev-secret
```

## Production（切替後・本番）
```
NEXT_PUBLIC_SITE_URL=https://mamasanlife.com
NEXT_PUBLIC_NOINDEX=false
NEXT_PUBLIC_GA_ID=G-XXXXXXX
NEXT_PUBLIC_ADSENSE_ID=ca-pub-XXXXXXXXXXXX

NEXT_PUBLIC_SANITY_PROJECT_ID=xxxx
NEXT_PUBLIC_SANITY_DATASET=production
SANITY_PROJECT_ID=xxxx
SANITY_DATASET=production
SANITY_API_VERSION=2024-03-14
SANITY_READ_TOKEN=（必要なら）
SANITY_WRITE_TOKEN=（必要なら）

LINE_BROADCAST_ENABLED=true
LINE_CHANNEL_ACCESS_TOKEN=（長期トークン）
SANITY_WEBHOOK_SECRET=（Webhook合言葉）
CRON_SECRET=prod-secret
```

> Vercel の Project Settings > Environment Variables で Preview/Production にそれぞれ設定し、デプロイごとに反映されます。
