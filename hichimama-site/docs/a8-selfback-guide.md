# A8.net セルフバック手順（Xserverドメイン想定）

この手順は Xserverドメインで `mama-3.com` を購入する際、セルフバック（自己アフィリエイト）で実質負担を下げるためのガイドです。

## 準備
- A8.net のアカウント作成/ログイン
- ブラウザの広告ブロッカーやトラッキング防止をオフ（Chrome 推奨）

## 手順
1. A8.net の「セルフバック」ページを開く
2. 検索で「エックスサーバー」「Xserverドメイン」を探す
3. 案件詳細を確認
   - 本人申込OK / セルフバック対象か
   - 成果対象（新規取得のみ・プレミアムドメイン対象外 など）
   - 還元条件（税抜◯◯円以上は◯◯円、未満は100% など）
   - Cookie 期限（クリックから◯日以内に申込）
   - 併用不可（他クーポン/他サイト経由はNGなど）
4. 「セルフバックを行う」を押して遷移
5. 同じタブのまま Xserverドメインで `mama-3.com` を購入
6. 申込完了画面のスクショ／注文番号／日時をメモ

## 承認まで
- 数日〜数週間で承認/否認が判定されます
- 承認状況は A8.net 管理画面で確認可能

## よくあるNG/注意
- 別タブで比較サイトを踏み直す（クッキー上書き）
- ブラウザや端末を跨ぐ（同一環境で完結推奨）
- 申込後にキャンセル/変更

## 代替案
- 案件が出ていない/条件が悪い場合
  - Xserver「レンタルサーバー」セルフバック＋「ドメイン永久無料特典」を使う
  - Porkbun/Dynadot の初年キャンペーンで直接安く購入（英語UI）
